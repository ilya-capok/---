 Распознавание мусора с использованием YOLOv8

 Описание проекта

В рамках практической работы разработан проект по распознаванию мусора на изображениях с использованием нейронной сети YOLOv8.

Цель работы

Изучение процесса обучения и применения современных моделей компьютерного зрения для автоматического обнаружения объектов.

Используемые технологии

* Python
* YOLOv8
* Ultralytics
* Kaggle Dataset TACO

Используемый датасет

TACO (Trash Annotations in Context)

Источник:
https://www.kaggle.com/datasets/ultralytics/taco

Структура проекта

* FailAI.py — код обучения модели;
* predict.py — код распознавания объектов;
* requirements.txt — используемые библиотеки;
* dataset/ — описание датасета;
* models/ — папка для хранения обученной модели.

 Вывод

В ходе выполнения практической работы был изучен процесс создания системы распознавания мусора на изображениях с использованием технологий машинного обучения и нейронных сетей.
